# NLP Final Project — LLM Routing

> 2026 Spring · Introduction to Natural Language Processing · Final Project

Given a user query, build a router that dynamically selects the most appropriate
model from a pool of 11 candidate LLMs — balancing answer performance against
cost. Submissions go to Kaggle; scoring metric is **Reward_{0.85}**.

**Deadline:** 06/26 (Friday) 23:59 — no late submission.

## Repository contents

- `data/train.csv` — ID, text query, and ground-truth performance (0.0–1.0) and cost for each of the 11 candidate models.
- `data/test.csv` — ID and text query only; predict the best model to route to.
- `data/sample_submission.csv` — required Kaggle submission format.
- `2026INLPFinalProject_LLMRouting.txt` — full project / task specification.
